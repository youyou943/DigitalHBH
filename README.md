# 数字黄宾虹(Digital Huang Binhong)

## 数字黄宾虹 是一个以现代数字技术为基础，结合传统艺术大师黄宾虹的画作风格，打造的数字艺术项目。通过数据可视化，本项目旨在探索黄宾虹艺术风格在数字平台上的创新再现，为传统艺术与数字艺术之间搭建一个桥梁。
